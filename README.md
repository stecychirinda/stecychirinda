## Featured AI/ML Project

### Income Eligibility Classification | Break Through Tech AI

**Problem:** A community-focused lender needed a more efficient and consistent way to screen applicants for income-based financial programs.

**What I did:** Developed and compared logistic regression and neural-network models using Census income data. I prepared the data, selected features, tuned model parameters, and evaluated accuracy, F1 score, interpretability, and fairness considerations.

**Result:** The neural network achieved **81.8% accuracy**, while logistic regression produced the stronger **0.680 F1 score**. I selected logistic regression because its performance was competitive and its predictions were easier to interpret in a high-impact eligibility context.

**Tools:** Python · pandas · NumPy · scikit-learn · Jupyter Notebook
